# field-ops-advisor# Project README

This repository contains both the backend (Node.js/Express) and frontend (Angular) for the project.

## Structure
- **backend/**: Node.js backend API
- **frontend/**: Angular frontend application

## Quick Start

### Backend
1. Navigate to the backend folder:
   ```bash
   cd backend
   npm install
   node index.js
   ```

### Frontend
1. Navigate to the frontend folder:
   ```bash
   cd frontend
   npm install
   ng serve
   ```
2. Open [http://localhost:4200](http://localhost:4200) in your browser.

## Development Notes
- Ensure Node.js is installed for both backend and frontend.
- Install Angular CLI globally for frontend: `npm install -g @angular/cli`
- Update API endpoints in frontend services to match backend routes.

## Folder Overview
- **backend/**: API, services, middleware, and data
- **frontend/**: Angular app, components, and assets
